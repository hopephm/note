GCD (Greatest Common Divisor) : 최대 공약수

	[방법]
	유클리드 호제법 : 자연수의 최대공약수를 구하는 알고리즘
	>> 나누어 떨어질 때까지 현재 나눈 값의 나머지를 구하고 해당 값으로 다시 나눔 (반복)

	Ex)

	930, 720

	930 % 720 != 0, 210
	720 % 210 != 0, 90
	210 % 90 != 0, 30
	90 % 30 == 0

	최대 공약수 = 30


	[원리]
	gcd(A, B) = G
		>> A = aG, B = bG 로 표현 가능
		>> A = Bq + r 이 되게하는 q에 대하여,
		>> aG = bGq + r 이므로

		r = G(a-bq)

		따라서 gcd(B,r) 역시 G를 최대 공약수로 갖는다.


LCM (Least Common Multiple) : 최소 공배수
	최소 공배수 = 두 자연수의 곱 / 최대 공약수
