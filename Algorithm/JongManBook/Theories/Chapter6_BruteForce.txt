1. 재귀호출
	<필수 요소>
	1) BaseCase(기저 사례) : 더 이상 나뉘지 않는 최소 작엄에 도달시 종료 조건
		* 기저사례를 모든 입력이 활용하여 계산될 수 있어야 함

	2) 재귀 호출 : 재귀 구조를 사용할 자기 자신의 함수 호출
		* 호출 결과를 사용하여 원하는 부분 동작을 풀 수 있어야 함

	3) 부분 문제 해결 : 호출 결과를 가지고 부분 문제를 해결하고 결과를 상위로 반환 


	<Tip>
	1) 중복제거 : 특정한 순서대로 답을 찾도록 강제 (순서대로 탐색하며 경우의 수를 찾아나감)
		* Ex) 조합
	2) 문제를 완전탐색으로 변형하기 : 논리 나열을 통해  문제를 다른 문제로 변형


	<문제 풀며 얻은 Tip>
	1) 중복 제거하는 방법을 논리적으로 정리 후, 코드 작성 시작할 것.
	2) 재귀형태로 시뮬레이션 할 경우, 
		< 세팅 -> 호출 - > 원상복귀 >
	   과정이 대부분
	3) map 형태 c++에서 그냥 string 입력으로 처리 가능
	4) result = 0 에서 재귀 결과를 더함으로써 상위 재귀 결과 리턴 가능
